# Vansh — Portfolio

A modern, accessible portfolio showcasing full-stack development work.

**Live Site:** [portfolio-red-ten-97.vercel.app](https://portfolio-red-ten-97.vercel.app)

## 🚀 Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, grid, flexbox, animations
- **Vanilla JavaScript** — No frameworks or dependencies
- **Tailwind CSS** — Via CDN for utility classes
- **Google Fonts** — Cormorant Garamond, Inter, JetBrains Mono

## ✨ Features

- Fully responsive design (320px to 4K)
- Dark theme with sophisticated color system
- Interactive terminal easter egg (press `/` or `` ` ``)
- Smooth scroll animations
- Custom cursor effects (desktop)
- WCAG 2.1 AA accessible
- Contact form integration
